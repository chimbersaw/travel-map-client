# Travel-map-client

https://map.yaytsa.com/

## Initialize

```
npm install
```

## Usage

### Development

Run the app in the development mode:

```
npm start
```

Open http://localhost:3000 to view it in the browser.

### Production

Create an optimized production build and serve it with a static server:

```
npm run build
npx serve -s build
```
